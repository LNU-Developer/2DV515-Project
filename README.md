# Getting Started

In order to start the recommendation system both the frontend and backend application needs to be started.
Data is seeded to an in memory SQL server on start.

## Available Scripts

In the FRONTEND project directory, you can run:
### `npm i`

Installs all needed dependencies.

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


In the BAKCEND project directory, you can run:

### `dotnet run`

Installs all needed dependencies. Runs the app in the development mode.\
Open [http://localhost:5001](http://localhost:5001) to view it in the browser.
